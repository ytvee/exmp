# Grid

The `Grid` component displays information.

## Usage

```jsx
import { Grid } from '@asi-create/design-system';

<Grid>
  <StyledItem xs={12} sm={6} md={4} lg={3}>
    Item 1
  </StyledItem>
  <StyledItem xs={12} sm={6} md={4} lg={3}>
    Item 2
  </StyledItem>
  <StyledItem xs={12} sm={6} md={4} lg={3}>
    Item 3
  </StyledItem>
  <StyledItem xs={12} sm={6} md={4} lg={3}>
    Item 4
  </StyledItem>
</Grid>;
```
