# Typography

The `Typography` component displays information.

## Usage

```jsx
import { Typography } from '@asi-create/design-system';

<Typography variant="h1" color="text.secondary">
  Headline1
</Typography>;
```

## Arguments

- **variant** _`HeadlineVariant | BodyVariant`_ - The variant of typography to apply. Possible values include 'h1', 'h2', 'h3', 'h4', 'h5' for headlines and 'body1', 'body2', 'body3', 'body4' for body text.
- **children** _`React.ReactNode`_ - The content to be rendered within the typography component.
